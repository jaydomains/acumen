# handovers/

One handover file per PR, written at PR close. Filenames should identify the
PR (e.g. `PR-001-<slug>.md`) and be derived from `HANDOVER_TEMPLATE.md` at the
repo root.

Handovers are immutable once written, except where business confidentiality,
privacy, or legal requirements compel an update.

This README is a directory placeholder so the otherwise-empty directory is
tracked by git. It is not a handover.
